# dsy teahouse

A lightweight static website for dsy teahouse — a Chinese-Japanese fusion tea space built around the traditions of *Chanoyu* and *Gongfu* tea culture.

## Stack

- HTML5
- CSS3 (Grid, Flexbox, custom animations)
- Vanilla JavaScript

## Structure

```
├── index.html      # Single-page site
├── style.css       # All styles
├── main.js         # Scroll animations, nav, hamburger menu
├── hero.jpg        # Hero background image
└── brief.md        # Original project brief
```

## Sections

- **Hero** — Full-screen atmospheric background with tagline and CTA
- **Philosophy** — Brand story covering the Chinese-Japanese tea fusion concept
- **Offerings** — Signature teas and space features in a responsive card grid
- **Visit Us** — Address, hours, contact info, and a stylised SVG map

## Features

- Responsive layout with mobile hamburger menu
- Scroll-triggered fade-in animations via IntersectionObserver
- Fixed nav with frosted-glass effect on scroll
- CSS steam animation on hero
- Rice paper and tatami texture via inline SVG patterns
